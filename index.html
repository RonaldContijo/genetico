Introdução

Os algoritmos genéticos(AGs) são uma classe de métodos de otimização e busca inspirados no processo natural de evolução descrito por Charles Darwin. Nos AGs soluções para o problema em questão são representadas como indivíduos, e esses individuos são submetidos a operadores de seleção, cruzamento e mutação para gerar as melhores caracteristicas evolutivas. Da mesma forma com que acontece na natureza, eiste um método de avaliação dessa população (grupo de individuos), e os mais adaptados (os que estão mais próimos de uma boa solução) possuem mais chanses de se reproduzir aeuadamente e passar seus genes (suas partes de solução) adiante.

Um problema clássico abordado pelos AGs é o da mochila e dos pesos. Nesse desafio, busca-se maimizar o valor dos itens que podem ser alocados em uma mochila, mas sem ultrapassar um peso limite que o saco pode suportar. Devido ao fato de ser um problema combinatório onde os passos da solução não importam (a resposta final não é um conjunto de passos, mas uma combinação de itens) ele pode ser resolvido através dos AGs.

Nesse ínterim, o presente trabalho tem como finalidade desenvolver uma página WEB com HTML (HyperText Markup Language), CSS (Cascading Style Sheets) e Javascript para demonstrar o desenvolvimento de uma implementação de GA para a resolução do problema da mochila, criar funções que permitam que o usuário modifique parametros de execução do algoritmo para verificar seus efeitos na resolução do problema e gerar gráficos informativos sobre os individuos no decorrer das iterações do algoritmo.

Esta pesquisa está dividida na conformação que se apresenta: (i) na Fundamentação Teórica são apresentadas as premissas encontradas na literatura que fundamentam a eistencia dos AGs. (ii) na seção Metodologia encontram-se as etapas da concepção do algoritmo; (iii) em Resultados e Discussões é exposto o protótipo desenvolvido na pesquisa; (iv) as derradeiras refleões estão presentes na seção de Considerações Finais.


Fundamentação Teórica

Charles Darwin, através de seus estudos da origem das espécies e de seu famoso livro "Sobre a Origem das Espécies por meio da Seleção Natural: A Preservação de Raças Favorecidas na Luta pela Vida", estabeleceu os princípios fundamentais da seleção natural, que serviram de base para o desenvolvimento dos Algoritmos Genéticos. A principal ideia que fundamenta os GAs é descrita por Darwin, C. (2003, p. 101).

...que os indivíduos mais aptos, entre os que sobrevivem, supondo que se produzem neles variações numa direção proveitosa, tendem a multiplicar-se em maior número que os indivíduos menos aptos.

Essa citação encapsula o princípio central desses algoritmos: a seleção natural. No contexto dos GAs, os "indivíduos mais aptos" se referem às soluções mais eficazes para um determinado problema, representadas por indivíduos em uma população de soluções candidatas. A ideia é que, ao longo das gerações, as soluções mais bem-sucedidas (mais aptas) têm maior probabilidade de sobreviver e se reproduzir, contribuindo para a evolução da população em direção a soluções melhores.

Russell e Norvig, em seu livro "Inteligência Artificial: Uma Abordagem Moderna" (2010, pg. 126-129), descrevem as etapas típicas de um algoritmo genético da seguinte maneira:

  1. Inicialização da População: Uma população inicial de soluções candidatas é gerada aleatoriamente ou de acordo com algum critério específico relacionado ao problema.
  2. Avaliação da Aptidão (Fitness): Cada solução na população é avaliada quanto à sua aptidão para resolver o problema. Isso geralmente envolve a atribuição de uma pontuação ou valor de aptidão com base em quão bem a solução atende aos critérios estabelecidos.
  3. Seleção: Indivíduos mais aptos na população são selecionados para reprodução com base em sua aptidão. Essa seleção pode ser realizada de várias maneiras, como seleção por torneio ou roleta.
  4. Reprodução: Os indivíduos selecionados são combinados (cruzamento ou crossover) para produzir descendentes que herdam características de seus progenitores.
  5. Mutação: Como Darwin escreveu em "A Origem das Espécies" (2003, p. 99), "...quando uma parte varia e a seleção natural acumula as variações, se originam, por vezes, outras modificações de natureza mais inesperada." A importância da mutação em algoritmos genéticos está relacionada à introdução de variabilidade na população de soluções candidatas. Assim como na natureza, onde a variação genética é boa para a evolução das espécies, a mutação em algoritmos genéticos permite explorar novas regiões do espaço de busca. Sem a mutação, a população poderia ficar presa, limitando a capacidade do algoritmo genético de encontrar soluções decentes para o problema em questão.
  6. Atualização da População: Os descendentes gerados substituem ou se juntam à população existente.
  7. Critério de Parada: O processo é repetido por várias gerações ou até que um critério de parada seja atingido. Isso pode ser um número máximo de gerações ou uma quando a solução ótima(a melhor solução) encontrada.

Metodologia

HTML, CSS e JavaScript são três tecnologias usadas para o desenvolvimento WEB moderno. O HTML é a linguagem padrão para a criação e estruturação de páginas web, definindo a estrutura do conteúdo por meio de elementos como cabeçalhos, parágrafos, imagens e links. CSS, Folhas de Estilo em Cascata, é utilizado para dar beleza aos elementos HTML, controlando aspectos artisticos como cores, fontes, layout e animações, permitindo uma apresentação visual bela, elegante e atraente em dispositivos de telas de tamanhos variados. JavaScript é uma linguagem de programação de alto nível que permite adicionar interatividade e dinamismo às páginas web, possibilitando a criação de funcionalidades como formulários interativos, animações, manipulação do DOM (Document Object Model). Essas três tecnologias trabalham para criar essa página. HTML para definir os elementos, CSS para estilizar, e Javascript para eecutar o AG desenvolvido e dar interatividade ao sítio.

Cálculo da Aptidão

A função desenvolvida examina uma mochila, representada por um vetor binário indicando a presença ou ausência de cada item, e calcula sua aptidão com base no peso e no valor total dos itens contidos nela. Primeiramente, a função percorre o vetor binário para determinar o peso e o valor acumulados dos itens presentes. Se o peso total excede o limite permitido, a aptidão é penalizada, multiplicando o valor da mochila por 0.1. Isso incentiva a busca por soluções com peso próximo ao limite máximo. Caso contrário, a aptidão é simplesmente o valor total dos itens presentes na mochila.

Métodos de seleção

Para selecionar os indivíduos para o cruzamento foram desenvolvidas duas funções. A função tournamentSelection realiza uma competição entre um número fixo de indivíduos (definido pelo tamanho do torneio) selecionados aleatoriamente da população. Cada indivíduo é avaliado com base em sua aptidão , e o vencedor do torneio é o que possui a maior aptidão. A função rouletteWheelSelection utiliza uma roleta ponderada para selecionar os indivíduos com base em sua aptidão relativa. Cada indivíduo tem uma fatia da roleta proporcional à sua aptidão, e um valor aleatório é gerado. O indivíduo cuja fatia inclui o valor gerado é selecionado. Ambas as funções buscam garantir que os indivíduos mais bem adaptados ao ambiente tenham maior probabilidade de serem selecionados, favorecendo a evolução da população ao longo das gerações. 


function tournamentSelection(population) {
    const tournamentSize = 5;
    const tournament = [];
    //seleção de 5 criaturinhas ao acaso para o torneio
    for (let i = 0; i < tournamentSize; i++) {
        tournament.push(population[Math.floor(Math.random() * population.length)]);
    }
    //retorna o melhor deles
    return tournament.reduce(
        (max, current) => (current[1] > max[1] ? current : max)
    );
}
function rouletteWheelSelection(population) {
    const totalFitness = population.reduce(
        (total, individual) => total + individual[1], 0
    );
    const randomValue = Math.random() * totalFitness;
    let currentSum = 0;
    for (const individual of population) {
        currentSum += individual[1];
        if (currentSum >= randomValue) {
            return individual;
        }
    }
}

Método de Mutação

A função mutate é responsável pela mutação de um cromossomo, representado pela mochila, em um algoritmo genético. A mutação é um processo para introduzir diversidade e evitar a convergência prematura para soluções não tão boas quanto a melhor das soluções. Nessa função, são gerados dois números aleatórios (pos1 e pos2) que representam as posições dos genes na mochila. Em seguida, os genes nas posições pos1 e pos2 são trocados entre si. Caso a mutação seja benéfica o indivíduo tera mais chances de se reproduzir e passar adiante seus genes. 




function mutate(mochila) {
    //gera 2 numeros aleatórios 
    //e realiza a troca de posição do que tem dentro de um com
    //o que ta no outro.
    const [pos1, pos2] = [
        Math.floor(Math.random() * NUMERO_ITENS),
        Math.floor(Math.random() * NUMERO_ITENS)
    ];
    [mochila[pos1], mochila[pos2]] = [
        mochila[pos2], 
        mochila[pos1]
    ];
    return mochila;
}
